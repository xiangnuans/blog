三、箭头函数

VS普通函数

  • 箭头函数只有一个参数时，括号可以参略
  • 只有一行return， {}可以参略
  • 作用域不同，箭头函数在哪个对象调用，this就是哪个对象

四、class 静态属性和实例属性

静态属性

指的是class本身的属性，即Class.propname，而不是定义在实例对象（this）的属性。
